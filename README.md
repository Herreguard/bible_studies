# Bible studies
This repository contains bible studies used within the friends group 'bende van ellende' and 'Vida'. Feel free to add your own! But: please keep the structure and naming conventions. Also: please stick to what the Bible says.

## Running locally

To preview the site with the GitHub Pages theme locally:

1. Install dependencies:
   ```bash
   bundle install
   ```

2. Start the Jekyll server:
   ```bash
   bundle exec jekyll serve --livereload
   ```

3. Open your browser to `http://127.0.0.1:4000`

The site will automatically reload when you make changes to your markdown files.
